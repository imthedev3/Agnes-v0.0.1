Quick Start
----------

Installation
~~~~~~~~~~~

.. code-block:: bash

   pip install agnes

Basic Usage
~~~~~~~~~~

.. code-block:: python

   from agnes import AgnesEngine

   engine = AgnesEngine()
   result = await engine.process_request(...)
